# Team Aqua Patch Cove (TAPC)
This is the repository for the Team Aqua Patch Cove; a website dedicated to hosting patching pages for ROMs in order to install ROMHacks made by members of the Team Aqua Hideout server. It is powered by the [RomPatcher.js](https://github.com/marcrobledo/RomPatcher.js) made by Marc Robledo.

If you wish to add your own hack, see [this tutorial](./tutorials/how_to_add_hack.md) for more info.
